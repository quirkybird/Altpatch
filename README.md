# AltPatch Monorepo

## Quick Start

```bash
pnpm install
pnpm dev
```

- extension mode health: http://127.0.0.1:7331/health
- extension build output: apps/extension/dist

## Vite Plugin Demo

```bash
pnpm --filter @apps/demo-vite-react dev
```

Open the Vite URL, then `Alt + Click` the demo button to see tooltip injection.
